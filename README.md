## Dev Scripts

`npm start`<br/>
deploys the client locally and watches for source changes.

`npm test`<br/>
runs tests.

`npm run build`<br/>
builds a production bundle.

`npm run format`<br/>
enforces source code formatting. This happens automatically on every commit.
